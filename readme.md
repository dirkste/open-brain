# Open Brain Client

A minimal web client for [Open Brain](https://github.com/NateBJones-Projects/OB1).

Captures thoughts to your personal Open Brain instance from any browser — phone, tablet, desktop, work machine. No app to install. Just a URL and your brain key.

## How it works

Open the URL, enter your brain key once, and start capturing. The key is validated against your Supabase instance before being stored locally. After that, one tap sends a thought directly to your Open Brain database.

## Usage

1. Host this file on GitHub Pages or any static host
2. Open the URL in any browser
3. Enter your brain key when prompted
4. Capture thoughts from anywhere
